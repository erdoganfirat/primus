                    ## Terminology

                    ### Tone
                    <p>Tone encompasses both the concepts of lightness and darkness within a color. It addresses the overall value of a color, regardless of its specific hue. The term "tone" is used to describe a color's placement along the lightness-darkness spectrum.</p>

                    ### Tint
                    <p>Tint describes a color achieved by adding white to the original hue. This process increases the lightness of the color while preserving its underlying hue. For example, the color pink is a tint of red.</p>

                    ### Shade
                    <p>Shade refers to a color that results from mixing black with the original hue. This mixture results in a darker color while retaining the initial hue. An instance of this is the color maroon, which is a shade of red.</p>

                    ### Hue 
                    <p>Hue is the characteristic that distinguishes a specific color from others on the color spectrum. It is the attribute that allows us to identify colors such as red, blue, or green.</p>

                    ### Saturation
                    <p>Saturation, also referred to as chroma or intensity, denotes the degree of purity or vividness of a color. A high level of saturation indicates a color that is vibrant and rich, while low saturation results in a more subdued or muted appearance.</p>

                    ### Lightness
                    <p>Lightness, known as value or brightness as well, signifies the perceptual attribute of a color related to its luminance. It defines the relative amount of light that a color reflects, leading to variations from dark to light. Higher lightness values correspond to colors that are closer to white, whereas lower values indicate colors that are closer to black.</p>


                    <div className="row row-cols-2">
                        <div className="col">
                            ### Color value 
                            <p>Color value is the alphanumeric code that accurately defines a specific color within various color systems or models.</p>
                        </div>
                        <div className="col">
                            <img src="color/colorvalue.png" srcset="color/colorvalue.png 1x, color/colorvalue@2x.png 2x" alt='Color value'/>

                    ### Hex code 
                    <p>A hex code is a six-digit code that represents colors in digital design and programming. It's used to define precise colors for websites, graphics, and more.</p>

                    ### Color token
                    <p>A specialized design token for color use. It refers to variable color values linked to themes. Color tokens are named by color and a number, like orange-50 for the #FF6000.</p>

                    ### Color index
                    <p>Color index is a numeric label used to identify and reference a particular color within a collection or system of colors.</p>

                    <img className="mb24" src="color/colorindex1.png" srcset="color/colorindex1.png 1x, color/colorindex1@2x.png 2x" alt='Color Index'/>

                    ### Contrast ratio
                    <p>The measurable contrast in brightness between two related color values, determined by applying the WCAG relative luminance formula.</p>

                    ### WCAG contrast minimum
                    <p>The WCAG (Web Content Accessibility Guidelines) contrast minimum refers to a specific requirement set by the WCAG to ensure that there is a sufficient contrast between text and its background on digital content, such as websites or applications. This requirement aims to enhance accessibility for people with visual impairments, making it easier for them to read and understand the content.</p>
                    <p>The WCAG defines specific contrast ratios between text and background colors that must be met to achieve sufficient readability. The contrast ratio is calculated by comparing the luminance values of the text and background colors. The minimum contrast ratio is typically expressed as a pair of numbers, such as 4.5:1 or 7:1, depending on the level of accessibility being targeted (such as AA or AAA).AA and AAA are levels of accessibility compliance. AA means basic accessibility, while AAA signifies a higher level of accessibility with more advanced features.</p>

                    ### Device color mode 
                    <p>A device color mode (also known as "mode") is a setting in the operating system that affects how the whole device looks. Most devices offer two options: device light mode and device dark mode. Primus Design System provides for both the device light mode and the device dark mode with accessibility features like high-contrast modes</p>

                    ### Color theme
                    <p>The Primus Design System encompasses three distinct color themes, each tailored to accommodate varying levels of visual acuity. These themes include the default, high contrast, and color blindness variants, each thoughtfully designed to cater to specific visual preferences and needs.</p>

                    ### Brand color palette
                    <p>The Primus Design System has ten unique brand color palettes. Even though Hepsiburada is the main brand, each subsidiary brand has its own identity. These brands use their primary colors for main interactions and share common colors for things like backgrounds, elevation, and text etc. The brand names in this system include Hepsiburada, Hepsipay, İş Ortağım, Hepsiburada Premium, Hepsiburada Market, Hepsiburada Çiçek, Hepsiburada Su, Hepsiglobal, Hepsiburada Seyahat, and Hepsiburada Oyun.</p>
